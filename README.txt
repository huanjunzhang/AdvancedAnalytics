The AdvancedAnalytics Package accompanies the book "The Art & Science of Data Analytics" by Dr. Edward Jones, Texas A&M Univ.

This package contains classes used to simplify data preprocessing and many machine learning algorithms.  The main classes are:

DecisionTree --> Creating and analyzing decision trees for both interval and classification targets

linreg --> Creating and analyzing linear regression models for interval targets

logreg --> Creating and analyzing logistic regression models for classification targets

NeuralNetowrk --> Creating and analyzing feed-forward neural networks for interval and classification targets.

News --> Web scraping news articles

ReplaceImputeEncode --> Data preprocessing and encoding for machine learning

Sentiment --> Sentiment analysis for documents and reviews based  upon the Afinn scoring system.

TextAnalytics --> Topic and cluster analysis for documents and reviews.
